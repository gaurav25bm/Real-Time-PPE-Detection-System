# 👷 Real-Time-PPE-Detection-System 

**Real-time Personal Protective Equipment (PPE) detection** using YOLOv8, OpenCV, and Streamlit. Detects safety gear like helmets and vests in live streams and uploaded media to ensure industrial safety compliance.

---

## 🚀 Live Demo
**Access the Web Application here:** https://real-time-ppe-detection-system-uyii4wlvxjfd2wvcx55jus.streamlit.app/

---

## ✨ Features 
- **Real-Time Detection**: Analyzes video frames to identify PPE usage instantly.
- **Upload Analysis**: Process static images or recorded video files for safety audits.
- **Optimized Performance**: Features frame-skipping inference for fluid monitoring.
- **Intuitive Interface**: Built with Streamlit for a clean, responsive user experience.

---

## 🛠️ Technologies 
- **Core AI**: Ultralytics YOLOv8
- **Image Processing**: OpenCV
- **Web Framework**: Streamlit
- **Deployment**: Streamlit Community Cloud
